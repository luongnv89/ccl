Installation

Available on PyPI — install with pip or uv, or use the one-line shell installer.

pip · recommended
$ pip install claude-codex-local
uv · alternative
$ uv tool install claude-codex-local
Or use the shell installer (no clone required)
bash <(curl -sSL https://raw.githubusercontent.com/luongnv89/claude-codex-local/main/install.sh)
Or install from source
git clone https://github.com/luongnv89/claude-codex-local.git
cd claude-codex-local
python3 -m venv .venv && source .venv/bin/activate
pip install -e .

Usage

Three steps. Ten minutes.
Local or cloud routing.

01

Install

Run one curl command. The wizard auto-detects your installed runtimes, checks your hardware, and flags anything missing.

02

Configure

Answer a few prompts. Pick your runtime, pick your model (or let llmfit pick for you). The wizard wires everything up safely.

03

Run

Type cc or cx for local. Use cc9/cx9 for 9router cloud routing. Your full Claude Code or Codex experience — every skill, every agent, every MCP server intact.

CLI Reference

Available Commands

ccl setup

Run the interactive setup wizard. Auto-detects your hardware, installed runtimes, and guides you through configuration.

Options:
--resume — Resume from last failed step
--non-interactive — Run without prompts (CI mode)

ccl doctor

Verify your configuration. Checks API keys, model availability, server connectivity, and configuration validity.

ccl find-model

Get hardware-aware model recommendations using llmfit. Analyzes your RAM and GPU to suggest models that will actually fit.

ccl serve

Start llama.cpp server with automatic restart on crashes. Useful for long-running inference sessions.

ccl run -p "prompt"

Non-interactive mode for scripted workflows. Execute a single prompt and exit.

Advanced

Configuration

Configuration Directory

All configuration lives in ~/.claude-codex-local/

  • → config.json — Main configuration file
  • → models.json — Model registry and metadata
  • → .9router_key — 9router API key (chmod 0600)
  • → .openrouter_key — OpenRouter API key (chmod 0600)

Shell Aliases

The wizard adds fenced blocks to your ~/.zshrc or ~/.bashrc

  • → cc / cx — Local model sessions
  • → cc9 / cx9 — 9router cloud routing
  • → cco / cxo / cpo — OpenRouter hosted routing

Environment Variables

Override defaults with environment variables

  • → CCL_OPENROUTER_MODEL — Override default OpenRouter model
  • → NO_COLOR — Disable colored output
