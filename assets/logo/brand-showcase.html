Brand Identity

CCL

Claude Code + Codex, running entirely on your machine — bridging cloud AI to local models.

Design Concept

Rounded square Codex app icon shape
8-ray starburst Claude's asterisk symbol
>_ prompt Codex terminal chevron + cursor

The mark is a direct fusion of Codex and Claude's visual languages. The rounded square app-icon shape comes from Codex. The 8-ray starburst radiating from the center is Claude's signature asterisk mark.

Layered on top — in neon green — is the `>_` terminal prompt: the chevron and cursor that Codex uses as its core symbol. Green signals "local", "active", "bridged" — the unique value CCL provides between the two cloud tools.

The result is a mark that developers instantly recognize as belonging to the Claude + Codex ecosystem, while the green `>_` makes clear this is the local, self-hosted variant.

The wordmark CCL in bold Inter is set at tight tracking — matching the no-nonsense density of CLI tooling.
